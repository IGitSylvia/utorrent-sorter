/**************************************uTorrentSorter v0.0.1**************************************/
To run the configuration manager double click the jar file or run:
javaw -jar "Drive:\Path\To\uTorrentSorter.jar"


to use set uTorrent to run 
javaw -jar "Drive:\Path\To\uTorrentSorter.jar" "%F" "%D" "%N" "%P" "%L" "%T" "%M" "%I" "%S" "%K"
when torrent changes state. Please note the qoute marks are important.

As of right now this program only sorts predefined labels when the torrents seeds
Labels supported for sort are:

-tv
-movie
-video
-music
-game
-document
